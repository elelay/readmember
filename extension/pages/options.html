Goodreads Integration

ReadMember requires that you register an application in Goodreads. Normally it's up to the application developper (me) to do it, but as all happens client-side, I don't want to entrust everybody with the secret key (if the application gets banned by somebody exploiting the credentials, everybody would be blocked).

That way, you don't have to trust me: review the code of the extension, self-host the data; and I don't have to trust you: the extension is registered at Goodreads to your name only.

Click Here to register a new application. You can leave the Application and Callback URL blank.
